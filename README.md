# RZN DSAG

자료구조 & 알고리즘 직접 구현해 보기 위한 리포지토리입니다.  
Visual Studio 2022 에서 C++로 작업하였습니다.  

RZNDSAG 솔루션에는 다음과 같이 두 프로젝트가 있습니다.
- **DSAG 프로젝트**
- **Main 프로젝트**

**DSAG 프로젝트**에 자료구조와 알고리즘들이 구현되어 있으며, 정적 라이브러리로 빌드됩니다.  
**Main 프로젝트**에서 필요한 헤더를 인클루드 한 후 사용하면 됩니다.
