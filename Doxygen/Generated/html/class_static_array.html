Detailed Description

template<typename T, std::size_t Size>
class StaticArray< T, Size >

정적 배열(고정 크기 배열, Static Array)을 캡슐화하는 컨테이너입니다.

요소의 개수가 Size 개인 T 형 배열을 래핑합니다.

◆ FillRange()

constexpr void StaticArray< T, Size >::FillRange ( const ElementType &  to_fill,
SizeType  range_start_index = DSAG_ARRAY_START_INDEX,
SizeType  range_last = Size 
)
inlineconstexpr

정적 배열의 range_start_index 이상 range_last 미만의 요소를 값 to_fill 로 채웁니다(Fill).

range_start_index 와 range_last 는 다음을 만족해야 합니다: \( 0 \leq \verb|range_start_index| < \verb|range_last| \leq \verb|Size| \)

Parameters
[in]to_fill설정할 값 입니다.
[in]range_start_index(Optional) 채울 범위의 시작 인덱스 입니다. 기본 값은 DSAG_ARRAY_START_INDEX 입니다.
[in]range_last(Optional) 채울 범위의 끝 입니다. 채울 범위의 마지막 인덱스에 1을 더한 값입니다. 기본 값은 Size 입니다.
